- Cache Aside Pattern（旁路缓存模式）
  Cache Aside Pattern 是我们平时日常业务开发中使用比较多的一个缓存读写模式，比较适合读请求比较多的场景。
  Cache Aside Pattern 中服务端需要同时维系 DB 和 cache，并且是以 DB 的结果为准。
  下面我们来看一下这个策略模式下的缓存读写步骤。
